# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка базы данных

1. Убедитесь, что MySQL запущен
2. Создайте файл `back/.env` со следующим содержимым:

```env
DB_HOST=localhost
DB_NAME=inhibitory_control
DB_USER=root
DB_PASSWORD=ваш_пароль
DB_PORT=3306
```

3. Инициализируйте базу данных:

```bash
python back/data_base.py
```

Или используйте автоматическую настройку:

```bash
python setup.py
```

## Шаг 3: Запуск сервера

```bash
python run_server.py
```

Сервер будет доступен по адресу: http://localhost:8000

## Шаг 4: Открытие сайта

Откройте файл `front/index.html` в браузере или используйте локальный сервер:

```bash
# Python
cd front
python -m http.server 8080

# Или просто откройте index.html в браузере
```

## Проверка работы

1. Откройте http://localhost:8000/docs для просмотра API документации
2. Откройте сайт и перейдите на страницу "Диагностика"
3. Пройдите тест и проверьте, что результаты сохраняются

## Решение проблем

### Ошибка подключения к базе данных
- Убедитесь, что MySQL запущен
- Проверьте параметры в файле `.env`
- Убедитесь, что база данных создана

### CORS ошибки
- Убедитесь, что backend запущен на порту 8000
- Проверьте, что в `back/main.py` настроен CORS middleware

### Файлы не загружаются
- Убедитесь, что вы открываете файлы через локальный сервер или используете правильные пути
- Проверьте консоль браузера на наличие ошибок
